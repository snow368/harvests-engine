========================
FIELD INFERENCE RULES
========================

Use visual reasoning to decide when a field must be filled vs. when unknown is acceptable.

camera_movement inference:
  If image contains: visible hand motion, motion blur, machine operation, moving ink, dynamic framing, unstable framing
  THEN camera_movement SHOULD NOT be unknown
  If image is: static product showcase, centered object, studio still image
  THEN camera_movement = static shot

shot_type inference:
  Product showcase, single object → macro close-up or medium close-up
  Tattooing process, body part visible → detail shot or machine close-up or needle perspective shot
  Artist working → artist POV or over-the-shoulder shot
  Studio environment → workstation shot or wide framing

camera_motion inference (motion section):
  No motion visible, static frame → static movement
  Handheld feel, slight shake → fast handheld motion
  Smooth gimbal/follow → smooth tracking motion
  Slow dramatic → slow cinematic motion

speed_feeling inference:
  Still image, product shot → still
  Active tattooing → real-time movement
  Slow deliberate motion → slow cinematic motion
  Fast needle work → fast handheld motion

vibration_feeling inference:
  Needle visibly operating → micro vibration or needle vibration
  No needle visible, product shot → none
  Static image → none

materials inference:
  No skin visible → skin_texture = unknown (valid)
  No gloves visible → glove_texture = unknown (valid)
  No ink visible → ink_reflection = unknown (valid)
  No plastic → plastic_reflection = unknown (valid)
  No metal → metal_reflection = unknown (valid)
  These unknowns are CORRECT — do not force-fill

needle_stage inference:
  No needle visible → unknown (valid)
  Needle touching skin → surface touch or micro penetration
  Needle fully in skin → full working depth
  Freshly pulled out → needle retracting
  Wiping area → wipe transition

skin_reaction inference:
  No skin visible → unknown (valid)
  Redness around tattoo → slight redness or heavy irritation
  Shiny skin → skin shine or vaseline reflection
  Clean unmarked skin → clean skin
  Stencil lines visible → fresh stencil remains
